# Basic Logic Gate    

## NOT   

### Boolean  
```
X = ¬A
```
## AND   
### Boolean  
```
X = A ∧ B
```
## OR     
### Boolean  
```
X = A ∨ B
```
## NAND  
### Boolean  
```
X = ¬(A ∧ B)
```
## NOR  
### Boolean  
```
X = ¬(A ∨ B)
```
## XOR  
### Boolean  
```
X = A ⊕ B = A¬B + ¬AB
```
## XNOR  
### Boolean  
```
X = ¬(A ⊕ B) = AB + ¬A¬B
```
